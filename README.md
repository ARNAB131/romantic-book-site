# For My Babe Girl

## Deploy on Vercel

1. Upload this folder to GitHub.
2. Import the GitHub repo in Vercel.
3. Use these settings:

Framework Preset: Other  
Build Command: npm run build  
Output Directory: .  
Install Command: npm install
